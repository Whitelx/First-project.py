# Digital Store Inventory System

A simple Python project for managing digital products, adding products to a shopping cart, calculating the total price, and saving cart data using JSON files.

## Features

- Display products from `products.json`
- Search for products by ID
- Add products to cart
- Calculate total cart price
- Save cart data to `cart.json`
- Save the user's name with the cart
- Basic error handling for missing data and invalid IDs

## Technologies Used

- Python
- JSON
- Git
- GitHub

## How to Run

1. Make sure Python is installed.
2. Open the project folder.
3. Run the program:

```bash
python main.py
```

or:

```bash
py main.py
```
